4th step

Create a list of players

After creating a database, you can now enter the players for their tournaments. Before you create a tournament, all players must be entered. You do this in the upper tab under Player List. Here are several ways to enter players: Add players from the DWZ list If you want to add players from the DWZ list, click the DWZ database button and add the players there. Adding players from the ELO list If you want to add players from the ELO list, click on the ELO database button and add the players there. Add Players Manually If you want to add players manually, click the New Player button and add the players there. After entering all players, you can now enter their tournaments.
