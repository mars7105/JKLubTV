JKlubTV

Version 4.0.61-beta

Manage chess round robin tournaments with JKlubTV

This application will capture the round-robin tournament belonging data into a SQLite database. It is intended for webmasters who want to easily manage a club tournament in their own chess club. The HTML tables that are to be published for the website will be created easily by the application. Automatically calculating the total points Sonneborn Berger points of each player, as well as their sequence DWZ, and sorts the HTML table by the calculated ranking. It is also possible the tables to save as a PDF file.
The software is free (opensource).
